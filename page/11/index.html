Some helpful Chrome tips

1. Unhide ‘Create application shortcuts’
*** Case study: add Slack shortcut instead of install Slack app on OS.
Từ version 44.0/2403.89, Google chrome thay thế tool ‘Create application shortcut’ bằng tool “Add to task bar”.
Tool ‘Add to task bar’, tương tự như tool ‘Create application shortcut’, cho phép người dùng add shortcut của trang web vào Google Apps (chrome://apps) và add vào task bar của hệ điều hành Windows. Tuy nhiên để sử dụng nó, bạn phải qua một vài thao tác.
